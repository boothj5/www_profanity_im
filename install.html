Overview

Profanity should run on any POSIX complaint operating system.

To run on Windows, a POSIX environment is required, such as Cygwin.

Arch and Gentoo currently have community supported packages.

Until distribution packages are created (volunteers welcome), a simple install script has been written which will work on Ubuntu and Fedora, the script installs dependencies and runs ./configure && make && make install

For all other distributions/operating systems the install script can be used as a guide to installing dependencies.

Installing on Linux

Arch

Profanity is available in the AUR:

https://aur.archlinux.org/packages/profanity/

If you prefer to keep up with the latest changes, a development version is also available at:

https://aur.archlinux.org/packages/profanity-git/

Instructions for installing from the AUR are at:

https://wiki.archlinux.org/index.php/AUR_User_Guidelines#Installing_packages

Gentoo

Profanity is available from the 'stuff' overlay. To install:

Add the "stuff" overlay:

layman -a stuff

or add to existing:

layman -s stuff

Install the package:

emerge profanity

Profanity can be linked with either libxml2 or expat libraries. By default expat is used, set the 'xml' flag to link with libxml2.

To install latest develepment version, add net-im/profanity to package.keywords file:

echo "=net-im/profanity-9999 **" >> /etc/portage/package.keywords
emerge =profanity-9999

Ubuntu

Download the latest source tarball, available here.

Extract the file and cd into the directory e.g:

tar -xvf profanity-0.2.1.tar.gz
cd profanity-0.2.1

Run the installer:

./install-all.sh

Fedora

If your user doesn't have Sudoer rights, add them to the list:

su
<enter password>
visudo

Find the line:

root ALL=(ALL) ALL

And add your user under it:

<username> ALL=(ALL) ALL

RPM

An RPM spec file is currently maintained by Florent Peterschmitt.

The spec file is available at the git repository here.

If libstrophe is not already on your system, a spec file is also available at the above location.

1. Setup

yum install gcc make expat-devel libstrophe

Fedora:

yum install fedora-packager
rpmdev-setuptree

CentOS:

mkdir -p ~/rpmbuild/{BUILD,RPMS,SOURCES,SPECS,SRPMS}

2. Build

cp profanity.spec rpmbuild/SPECS
cp profanity-<version>.tar.gz rpmbuild/SOURCES
rpmbuild -bb rpmbuild/SPECS/profanity.spec

3. Install

yum localinstall
rpmbuild/RPMS/<arch>/profanity-<version>.<dist>.<arch>.rpm

or, if localinstall is not recognised:

rpm -i rpmbuild/RPMS/<arch>/profanity-<version>.<dist>.<arch>.rpm

Other distros

For other linux distributions, the install script is a good guide. You will need to install the development packages for dependencies, which are:

libssl, expat, ncursesw, glib2.0, libnotify, libcurl and libXScrnSaver.

Profanity uses the libstrophe XMPP library which must be downloaded and built from source which can be found at Github.

Installing on Windows

You will need Cygwin to run profanity on Windows:

http://www.cygwin.com

If you don't already have it, install Cygwin by downloading and running:

http://cygwin.com/setup-x86_64.exe

You will need git, wget and unzip to install Profanity which are not by default installed with Cygwin. When running setup.exe (or running it again to install these packages) you can search for them and add them to the install when you are asked to select packages.

Download the latest source zip, availavle here.

Put the zip somewhere on your Cygwin drive for example:

C:\cygwin\home\james

Unzip the file and cd into the directory e.g:

unzip profanity-0.2.1.zip
cd profanity-0.2.1

Run the installer:

./install-all.sh
