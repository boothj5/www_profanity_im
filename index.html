Profanity

A console based XMPP client

Profanity is a console based XMPP client written in C using ncurses and libstrophe, inspired by Irssi

Latest release: 0.1.9
Download: profanity-0.1.9.tar.gz

Installation instructions and FAQ can be found on the Github Wiki

Mailing list: Google Groups Profanity

The source can be found on Github

Features

  • Supports any XMPP chat service including Google Talk
  • Configuration via a simple properties file
  • Desktop notifications and reminders
  • Incoming Typing notifications
  • Dated chat logs by user
  • Option to show history of previous messages with a user
  • Ability to send Tiny URLs
  • Customisable UI
